# 🌊 Aquatic Science Portfolio Website

A modern, professional one-page portfolio website designed for aquatic science experts, featuring a beautiful oceanic theme with shades of blue and aqua.

## 🐠 Features

### Design & Theme
- **Oceanic Theme**: Calm, professional design with blue and aqua color palette
- **Modern Layout**: Clean, single-page design with smooth navigation
- **Responsive Design**: Fully mobile-responsive for all device sizes
- **Water Effects**: Animated SVG waves, floating bubbles, and swimming fish animations
- **Professional Typography**: Elegant font combinations (Inter + Playfair Display)

### Sections Included
1. **Home (Hero)** - Introduction with animated elements and call-to-action buttons
2. **About Me** - Professional background with animated statistics
3. **Research & Projects** - Showcase of key research areas with visual placeholders
4. **Skills & Tools** - Categorized technical skills with interactive tags
5. **Certifications** - Professional certifications with icons and dates
6. **Publications** - Academic publications with metrics and institution logos
7. **Contact** - Contact information and functional contact form

### Interactive Features
- ✅ **CV Download Button** - Downloads a formatted CV file
- ✅ **Research Photos/Data Graphs** - Placeholder areas for visual content
- ✅ **Institution Logos** - Space for affiliated organization logos
- ✅ **Smooth Scrolling Navigation** - Seamless section transitions
- ✅ **Mobile Menu** - Hamburger menu for mobile devices
- ✅ **Contact Form** - Functional form with validation
- ✅ **Scroll Animations** - Elements animate as they come into view
- ✅ **Counter Animations** - Statistics animate when visible
- ✅ **Hover Effects** - Enhanced interactivity throughout

### Technical Implementation
- **Pure HTML5/CSS3/JavaScript** - No framework dependencies
- **Modern CSS Features**: Grid, Flexbox, Custom Properties, Animations
- **Intersection Observer API** - For scroll-triggered animations
- **Mobile-First Design** - Responsive breakpoints for all screen sizes
- **Performance Optimized** - Throttled scroll events and efficient animations
- **Accessibility** - Keyboard navigation and screen reader friendly

## 🚀 Getting Started

### Simple Setup
1. Download all files to a folder
2. Open `index.html` in any modern web browser
3. The website works immediately - no installation required!

### File Structure
```
portfolio/
├── index.html          # Main HTML file
├── styles.css          # CSS styles and animations
├── script.js           # JavaScript functionality
└── README.md           # This documentation
```

## 🎨 Customization

### Content Updates
1. **Personal Information**: Update name, title, and contact details in `index.html`
2. **About Section**: Modify background story and statistics
3. **Research Projects**: Replace placeholder content with actual research
4. **Skills**: Update skill categories and tags
5. **Certifications**: Add your specific certifications
6. **Publications**: Replace with your actual publications

### Visual Customization
1. **Colors**: Modify CSS custom properties in `styles.css`
2. **Images**: Replace placeholder divs with actual photos
3. **Logos**: Add real institution logos
4. **Typography**: Change fonts by updating Google Fonts links

### Advanced Features
1. **Real Form Submission**: Integrate with form services like Formspree or Netlify Forms
2. **Image Gallery**: Add lightbox functionality for research photos
3. **Blog Integration**: Add a blog section for research updates
4. **Analytics**: Add Google Analytics tracking
